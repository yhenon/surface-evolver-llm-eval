How it works

A real artifact boundary, not a prose answer.

A correct answer must load in Surface Evolver, evolve under hidden scripts, and land expected geometry measurements within tolerance.

Generate

Models receive a task prompt plus tools for curated Evolver documentation, trial execution, and final structured `.fe` submission.

Repair

The runner allows bounded feedback rounds so models can inspect Evolver errors and submit a revised datafile.

Grade

Static datafile checks, hidden Evolver runs, and metric checks combine into a weighted score with a strict pass/fail label.
